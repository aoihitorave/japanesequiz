インストール方法

📱 スマートフォンへのインストール

iPhone/iPad (iOS Safari)

  1. Safariでこのページを開く
  2. 画面下部の共有ボタン □↑ をタップ
  3. 「ホーム画面に追加」を選択
  4. 名前を確認して「追加」をタップ

Android (Chrome)

  1. Chromeでこのページを開く
  2. 右上のメニュー ⋮ をタップ
  3. 「ホーム画面に追加」または「アプリをインストール」を選択
  4. 「インストール」をタップ

💻 パソコンへのインストール

Windows/Mac (Chrome/Edge)

  1. Chrome または Edge でこのページを開く
  2. アドレスバー右側のインストールアイコン ⊕ をクリック
  3. 「インストール」をクリック
  4. デスクトップやスタートメニューからアプリを起動可能

🔄 アプリの更新方法

自動更新

アプリを開くと自動的に最新版をチェックし、バックグラウンドで更新されます。

手動更新

  • スマートフォン：画面を下にスワイプして更新
  • パソコン：Ctrl+F5 (Mac: Cmd+Shift+R)

🗑️ アンインストール方法

iPhone/iPad

  • ホーム画面のアプリアイコンを長押し
  • 「ブックマークを削除」または「×」マークをタップ

Android

  • ホーム画面のアプリアイコンを長押し
  • 「アンインストール」を選択

Windows

  • スタートメニューでアプリを右クリック
  • 「アンインストール」を選択

Mac

  • Launchpadでアプリアイコンを長押し
  • ×マークをクリック

アプリ情報

📖 このアプリについて

日本語の美しい表現を楽しく学べる学習アプリです。 子供から大人まで、幅広い年齢層の方にご利用いただけます。

  • ことわざ・慣用句・故事成語を収録
  • 難易度別の問題で段階的に学習
  • サドンデスモードで腕試し
🔒 完全オフライン・個人情報収集なし

🔒 プライバシーについて

本アプリは完全にオフラインで動作し、個人情報の収集・送信は一切行いません。

  • 学習履歴は端末内にのみ保存
  • 外部サーバーとの通信なし
  • 広告・トラッキングなし
  • アカウント登録不要

安心してご利用いただけます。

📚 収録内容

各カテゴリーに難易度別の問題を収録しています。

  • ことわざ：生活の知恵や教訓を含む伝統的な表現
  • 慣用句：日常会話で使われる特別な意味を持つ表現
  • 故事成語：中国の歴史や文学に由来する四字熟語など

🎮 学習モード

  • 初級・中級・上級：レベルに応じた10問のクイズ
  • ランダム：全難易度から無作為に10問出題
  • サドンデス：1問でも間違えたら終了の緊張感あるモード
